BaristaLang

By Simmon Quan

About the language:

BaristaLang is a python based scripting language that aims to emulate a conversation a customer would have with a barista at a cafe. The goal of baristaLang is to have an easy to learn, human-speech like syntax that will hopefully get beginners to start coding.

Features ~ BaristaLang can interpret:

  • user provided math equations with operator precedence
  • if statements
  • print statements
  • FizzBuzz
  • prompts for websites
  • prompts for ASCII art
